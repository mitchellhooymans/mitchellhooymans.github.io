King (1966) Profile Viewer

Preview tidally truncated cluster density profiles before you run a simulation

King (1966) model

This tool solves the Poisson equation for the King (1966) model, which describes the density profile of relaxed, tidally truncated star clusters (such as globular clusters). The dimensionless central potential W0 sets the shape; scale parameters r0, half-mass radius rh, or virial radius rv map the profile to physical units.

Density ρ0 vs your N-body setup. The curve is a smooth mass density ρ(r) (fluid limit). In simulations you usually set N, an IMF (stellar masses), and structural scales (e.g. virial radius, rh, tides) — not ρ0 directly. Total mass is M ≈ N × ⟨m⟩IMF; the IMF sets how that mass is split among stars, while ρ(r) sets where they sit. Here ρ0 only rescales the vertical axis (same shape if you change W0 or radii). Use this page to preview structure; use N + IMF + radii when you build ICs.

Virial radius. For each W0, rv is found from the solved King model (local virial balance M(<r)/r ≈ ⟨v2⟩), with rh/rv ≈ 0.78 for concentrated models (W0 ≳ 5), rising toward ~0.95 when W0 is small — close to the ~0.78 factor used in cluster IC tools at typical W0.
